USAGE EXAMPLE
================================================================================

1. Run: python starkmann_email_processor.py

2. GUI window opens with:
   ┌───────────────────────────────────────┐
   │ Starkmann Email to Excel Processor   │
   ├───────────────────────────────────────┤
   │ Input File:  [Browse...]              │
   │ Output File: [Browse...]              │
   │ [Process & Export]  [Exit]            │
   │                                       │
   │ Information:                          │
   │ ...                                   │
   └───────────────────────────────────────┘

3. Click "Browse..." and select email file (e.g., "25-Nov-24.txt")
   Output automatically set to: "2024_11_25.xlsx"

4. Click "Process & Export"

5. Two files created:
   - 2024_11_25.xlsx (Excel file with book data)
   - 2024_11_25.books (Raw data file)
